## Python Code Quality Tools Review

This repo is for testing out code quality tools available in Python. The requirements.txt lists the tools and their versions, but it's recommended to run them using `pipx`/`uvx` to avoid installing into your environment.

The code intentionally has errors to test out the following code quality tools:

### Linters

- ruff (check)
- pylint
- flake8

### Formatters

- black
- ruff (format)
- autopep8

### Type Checkers

- mypy
- pyright
- pyrefly
- ty
